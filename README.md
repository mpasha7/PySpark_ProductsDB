# PySpark_ProductsDB
С помощью фреймворка PySpark импортируются данные (из CSV-файлов в pyspark.sql.dataframe) о Продуктах, их Категориях и Связях между ними (многие ко многим).
Реализован метод, связывающий данные из этих датафреймов по принципу LEFT JOIN + LEFT JOIN таким образом, что в итоговом датафрейме присутствуют все пары "Имя продукта - Имя категории" и имена продуктов без категорий.
Категории, не принадлежащие ни к одному продукту в выборку не попадают.
